# Giap Nguyen

I am a vehicle control and state-estimation engineer based in South Korea. My work focuses on model-based control, vehicle dynamics, Kalman-filter-based estimation, nonlinear optimization, and implementation-oriented validation.

## Technical Interests

- Vehicle control and motion-control algorithms
- Model predictive control and constrained optimization
- Vehicle state estimation and parameter identification
- GNSS/INS navigation and error-state Kalman filtering
- MATLAB/Simulink prototyping and C/C++ implementation validation

## Repository Notes

Most repositories here are research or study-oriented implementations of control and estimation algorithms. Some reproduce methods from published papers, while others are simplified examples used to understand, validate, and communicate algorithm behavior.

They are not presented as production software. The main purpose is to show algorithm understanding, implementation practice, and validation workflow.
